# SKILL.MD --- Design de Header e Sidebar (UI Moderna)

## 1. Objetivo

Criar apenas o design de um Header e uma Sidebar modernos, genéricos e
reutilizáveis, onde o usuário poderá escolher quais páginas e
informações adicionar depois.

O design deve ser moderno, profissional, com aparência de sistema
premium (Fintech / SaaS / Plataforma).

------------------------------------------------------------------------

## 2. Tecnologias

Utilizar:

-   HTML
-   CSS (Tailwind CSS ou CSS Modules)
-   JavaScript

Bibliotecas para efeitos: - GSAP (animações) - Lucide (ícones) - Vanilla
Tilt (efeito 3D suave) - CSS Animations - Hover Effects -
Glassmorphism - Gradientes animados

------------------------------------------------------------------------

## 3. Header (Barra Superior)

### Estrutura do Header:

O Header deve conter:

-   Fundo semi-transparente (Glassmorphism)
-   Efeito de desfoque (backdrop blur)
-   Título no lado esquerdo:\
    **Dashboard \| Nome da Página**
-   Lado direito:
    -   Foto de perfil do usuário
    -   Nome do usuário
    -   Botão de alternar tema Dark / Light
    -   Ícones

### Estilo do Header:

-   Deve parecer que está "flutuando" sobre o background
-   Fundo com transparência
-   Borda sutil
-   Sombra suave
-   Efeito glow suave
-   Animação suave ao aparecer
-   Responsivo

### Efeitos no Header:

-   Glassmorphism
-   Glow suave
-   Hover nos botões
-   Animação ao trocar tema
-   Microinterações nos ícones
-   Shimmer effect sutil
-   Transição suave ao rolar a página

------------------------------------------------------------------------

## 4. Sidebar (Menu Lateral)

### Estrutura da Sidebar:

A Sidebar deve conter:

-   Logo no topo (imagem enviada pelo usuário)
-   Nome "Performance" ao lado a logo
-   Menu com ícones
-   A Sidebar deve ser recolhida (apenas ícones)
-   Ao passar o mouse, ela expande e mostra o nome das páginas
-   Itens do menu com ícones + nome da página
-   Área inferior com configurações ou perfil

### Comportamento:

-   Sidebar recolhida por padrão
-   Expandir ao passar o mouse
-   Animação suave ao expandir
-   Ícones com animação ao passar o mouse
-   Item ativo deve ficar destacado
-   Deve ser responsiva (no mobile vira menu hambúrguer)

------------------------------------------------------------------------

## 5. Efeitos Visuais da Sidebar

A Sidebar deve ter:

-   Efeito de LED nas bordas (gradiente animado mudando de cor)
-   Glow nas bordas
-   Animação na logo (movimento suave/flutuação)
-   Hover moderno nos itens
-   Efeito de luz passando (shimmer)
-   Efeito Tilt 3D muito suave
-   Sombra suave
-   Glass effect
-   Gradiente animado na borda
-   Microinterações ao clicar
-   Animação de entrada dos itens

------------------------------------------------------------------------

## 6. Interações

### Interações obrigatórias:

-   Hover nos itens do menu
-   Item ativo destacado
-   Sidebar expandindo ao passar o mouse
-   Animação na logo
-   Botão de tema com animação
-   Ícones com animação
-   Transições suaves
-   Cursor interativo (opcional)
-   Tooltip quando a sidebar estiver recolhida
-   Feedback visual ao clicar

------------------------------------------------------------------------

## 7. Responsividade

Regras:

-   Em telas pequenas:
    -   Sidebar vira menu hambúrguer
    -   Header se adapta
    -   Nome do usuário pode sumir e ficar só a foto
    -   Título se adapta
-   Nada pode sobrepor outro elemento
-   Tudo deve se reorganizar automaticamente

------------------------------------------------------------------------

## 8. Estilo Visual

O design deve utilizar:

-   Glassmorphism
-   Glow
-   Gradientes
-   Bordas iluminadas
-   Sombras suaves
-   Transparência
-   Blur
-   Neon
-   Microinterações
-   Animações suaves
-   Aparência moderna e tecnológica

------------------------------------------------------------------------

## 9. Qualidade do Design

O design NÃO pode:

-   Ser simples demais
-   Parecer template genérico
-   Ter animações bruscas
-   Ter elementos desalinhados
-   Ter cores sem harmonia

O design DEVE:

-   Parecer profissional
-   Parecer sistema premium
-   Ter profundidade visual
-   Ter camadas visuais
-   Ter animações suaves
-   Ter identidade visual
-   Ter consistência visual

------------------------------------------------------------------------

## 10. Regra Final

O Header e a Sidebar devem parecer parte de um sistema profissional
real. O design deve focar em:

-   Experiência do usuário
-   Animações suaves
-   Interface moderna
-   Interatividade
-   Aparência premium
