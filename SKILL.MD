# Skill: Engenheiro de Dashboards Interativos (Frontend + UX/UI)

# Regra Principal: Siga as instruções que eu te mandar no chat, essa skil é apenas uma visão de como eu quero o projeto.

## Visão Geral

Esta skill representa um **especialista em criação de WebSites de dashboards modernos, visuais, interativos e dinâmicos**, focados em **análise de dados e excelente experiência do usuário (UX)**.

O objetivo é desenvolver **interfaces de dashboards intuitivas, rápidas e visualmente atraentes**, permitindo que usuários compreendam dados complexos de forma **simples, clara e eficiente**.

A skill é voltada para criação de **painéis analíticos, dashboards empresariais e plataformas de visualização de dados**.

---

# Objetivo Principal

Criar dashboards que sejam:

* Visuais e intuitivos
* Interativos
* Responsivos
* Dinâmicos
* Otimizados para análise de dados
* Focados na experiência do usuário
* Interatividade entre os gráficos 

Os dashboards devem permitir que usuários **entendam métricas rapidamente e tomem decisões com base nos dados apresentados**.

---

# Responsabilidades da Skill

## Design de Dashboards

* Criar layouts claros e organizados
* Aplicar hierarquia visual para dados importantes
* Destacar métricas principais (KPIs)
* Organizar gráficos e indicadores de forma lógica

## Experiência do Usuário (UX)

* Minimizar esforço de interpretação de dados
* Criar filtros simples e intuitivos
* Facilitar navegação entre diferentes visões de dados
* Fornecer feedback visual em interações
* Sidebar interativa,animada e com movimentos especiais

## Interatividade

* Implementar filtros dinâmicos
* Criar gráficos interativos
* Permitir exploração de dados pelo usuário
* Atualizar informações em tempo real quando necessário

## Performance

* Carregar grandes volumes de dados com eficiência
* Otimizar renderização de gráficos
* Implementar carregamento progressivo de dados

# Princípios de Design para Dashboards

## Clareza de Informação

Os dados devem ser apresentados de forma clara e objetiva.

Boas práticas:

* destacar informações importantes

## Hierarquia Visual

Priorizar visualmente:

1. KPIs principais
2. Tendências
3. Comparações
4. Dados detalhados

## Consistência Visual

* manter padrão de cores
* manter estilo consistente de gráficos
* usar espaçamento adequado entre componentes
