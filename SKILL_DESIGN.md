# SKILL.MD --- Padrão de Design para Sites Modernos (Dark/Light, Efeitos, UI/UX)

## 1. Objetivo

Criar um site com aparência moderna, profissional e diferenciada,
evitando aparência de sites genéricos feitos por IA. O site deve ter
aparência de sistema premium (Fintech / SaaS / Plataforma Profissional).

------------------------------------------------------------------------

## 2. Tecnologias Obrigatórias

O site deve ser desenvolvido utilizando:

-   HTML
-   CSS
    -   Tailwind CSS ou CSS Modules ou Styled Components
-   JavaScript

Bibliotecas obrigatórias para efeitos: - GSAP (animações) - tsParticles
(partículas interativas) - Lucide (ícones modernos) -  - ScrollTrigger (animações ao rolar) - Rellax.js (Parallax) - Lottie
(animações) - Three.js (quando necessário para 3D)

------------------------------------------------------------------------

## 3. Tema do Site

O site deve possuir:

-   Tema Dark (principal)
-   Tema Light (opcional com botão de alternância e efeito de visual de alternância)
-   Background com imagem escolhida pelo usuário
-   Efeitos animados no fundo (partículas, formas, gradientes animados,
    etc.)
-   As cores do site devem combinar com o background
-   O site deve ter aparência moderna e tecnológica

------------------------------------------------------------------------

## 4. Estilo Visual

O site deve utilizar:

-   Glassmorphism (efeito de vidro)
-   Glow neon
-   Sombras suaves
-   Brilho
-   Gradientes
-   Transparências
-   Bordas suaves
-   Profundidade de camadas
-   Design estilo fintech premium
-   Neumorphism
-   Shimmer effect

------------------------------------------------------------------------

## 5. Efeitos Obrigatórios

O site deve conter efeitos modernos como:

-   Partículas interativas no fundo
-   Elementos flutuantes
-   Animações GSAP suaves
-   Glow neon
-   Glassmorphism
-   Sombra suave
-   Brilho
-   Cursor que segue o mouse
-   Cursor interativo
-   Parallax
-   Scroll animations
-   Hover effects (microinterações)
-   Elementos surgindo ao rolar a página
-   Liquid effects
-   Background animado
-   Gradientes animados
-   Loading animations
-   Transições suaves entre páginas
-   Animações ao clicar
-   Animações ao passar o mouse

Os efeitos devem ser suaves e profissionais, sem exagero.

------------------------------------------------------------------------

## 6. Ícones

O site deve utilizar apenas ícones modernos. Utilizar a biblioteca
Lucide. Não utilizar emojis.

------------------------------------------------------------------------

## 7. UI / UX

Regras de interface:

-   Layout limpo
-   Interface intuitiva
-   Hierarquia visual clara
-   Espaçamento generoso
-   Títulos grandes
-   Subtítulos suaves
-   Textos bem alinhados
-   Botões com animação
-   Feedback visual ao usuário
-   Navegação fácil
-   Design consistente em todas as páginas
-   Elementos bem organizados
-   Nada pode ficar sobreposto

------------------------------------------------------------------------

## 8. Responsividade

O site deve ser responsivo e funcionar em:

-   Desktop
-   Notebook
-   Tablet
-   Celular

Regras: - Sidebar vira menu hambúrguer no mobile - Elementos se
reorganizam automaticamente - Nenhum elemento pode sobrepor o outro -
Textos e títulos devem se adaptar - Botões devem funcionar no celular -
Layout deve continuar bonito em qualquer tela

------------------------------------------------------------------------

## 9. Qualidade Visual

O site NÃO pode:

-   Parecer site simples
-   Parecer template genérico
-   Ter cores sem harmonia
-   Ter elementos sobrepostos
-   Ter animações travando
-   Ter espaçamento ruim
-   Ter fonte feia
-   Ter layout desorganizado

O site DEVE:

-   Parecer profissional
-   Parecer sistema premium
-   Ter profundidade visual
-   Ter camadas visuais
-   Ter microinterações
-   Ter animações suaves
-   Ter identidade visual
-   Ter consistência visual
-   Ter aparência moderna

------------------------------------------------------------------------

## 10. Estrutura do Site

O site deve possuir:

-   Header (barra superior)
-   Sidebar (menu lateral)
-   Área principal (conteúdo)
-   Cards
-   Botões
-   Modais
-   Tooltips
-   Área de perfil

------------------------------------------------------------------------

## 11. Regra Final

Sempre desenvolver o site pensando como:

-   Web Designer
-   UI Designer
-   UX Designer
-   Desenvolvedor Frontend

O foco deve ser experiência do usuário, design moderno e interatividade.
